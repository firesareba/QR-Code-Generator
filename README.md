# QR-Code-Generator
## MADE FROM SCRATCH. NO LIBRARIES USED.
A QR Code Generator that goes direct to the site. No intermediate server or expiration date.
## Functional Features
### Mask
Flips certain bits to make the QR Code easier for Scanner to understand. This website offers all 8 Mask Patterns.
### Error Correction
Error Correction accounts for if your QR Code is damaged or misread by the Scanner. This way, it can still scan even if there is a stain or something. This website offers all 4 levels of Error Correction. With higher correction levels, you give up more data space but gain durability.
### Version
There are 40 QR Code Versions, the main difference being the size. Just 26 Version 40 QR Codes could store all of Apollo 11's computer's programming. For most use cases, you will use QR Codes between Version 2 and 10.
## Stylistic Features
### Download
Easily Download the QR Code as an image. You can email, print, and embed it with ease.
### Logo
Add a your logo to the center of the Code. Note this might not work as well with smaller QR Codes.
### Color
Customize the colors used in your QR Codes. I would recomend to make the 2 colors contrast each other so the Scanner understands it easily.
